# Timetable Generator

A MERN stack application to generate class and teacher timetables, ensuring balanced teacher load and no room conflicts.

## Features

- Generate timetables for classes and teachers
- Ensure balanced load among teachers
- Avoid room conflicts
- User-friendly interface with responsive design

## Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js

## Prerequisites

- Node.js and npm installed
- MongoDB installed and running

## Usage

1. Click on the "Generate Timetables" button to create timetables for classes and teachers.
2. View the generated timetables for both classes and teachers.
